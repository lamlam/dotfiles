# My Dotfiles

## setup

clone this repository and exec setup script

```bash
./script/setup.bash
```

### Homebrew

Homebrew-related scripts need to be executed separately.

```bash
# insatll
/bin/bash script/setup-homebrew.bash

# backup
/bin/bash script/dump-homebrew.bash
```
